As part of the Master Setup steps a file with this name should be written to the workspace containing json description of all the tasks to execute. Example:

[
  {
    "name": "Task1",
    "env": {
      "VAR1": "VALUE1",
      "VAR2": "VALUE2"
    }
  },
  {
    "name": "Task2",
    "env": {
      "VAR1": "VALUE1",
      "VAR2": "VALUE2"
    }
  }
]
